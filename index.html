睡眠癒るり プライバシーポリシー

最終更新日：2025年11月23日

開発者：癒るり369

本プライバシーポリシーは、モバイルアプリ「睡眠癒るり」（以下、「本アプリ」）において、 利用者の情報がどのように取り扱われるかを定めるものです。

1. 取得する情報

本アプリでは、次の情報を取得する場合があります。

  • お問い合わせに関する情報
    本アプリ内の「運営メール」から、利用者ご自身のメールアプリを通じてお問い合わせを行う場合、 メール本文に含まれる情報（メールアドレス、本文、添付ファイルなど）は、 利用者が使用しているメールサービスを通じて送信されます。
    本アプリ自体は、メール本文や連絡先等をアプリ内部で収集・保存する処理は行いません。
  • 広告に関する情報
    本アプリでは、Google LLC が提供するモバイル広告 SDK（AdMob 等）を利用して広告を表示します。
    広告表示のため、以下の情報が Google などの広告配信事業者により収集・送信される場合があります。
    • 広告 ID（AAID）
    • 端末情報（OS バージョン、端末の種類、言語設定など）
    • 広告の表示・クリック等に関する情報
    なお、これらの情報は広告配信事業者が取得・利用するものであり、 開発者が当該情報を取得・保存することはありません。
  • 通知に関する情報
    本アプリでは、BGM やタイマー機能のために端末の通知機能を利用します。
    通知は端末内で完結して動作し、開発者が管理する外部サーバー等へ送信されることはありません。
    （将来、プッシュ通知等を導入する場合は、本ポリシーを更新します。）

2. 情報の利用目的

取得した情報は、主に次の目的で利用します。

  • 本アプリの提供・運営・改善のため
  • お問い合わせへの回答や必要な連絡のため
  • 広告の表示・最適化（広告機能を有効にしている場合）
  • 不具合対応・セキュリティ対策など、サービスの安全な運用のため

3. 第三者提供について

本アプリの開発者が、利用者の個人情報を本人の同意なく第三者に提供することはありません。 ただし、次に該当する場合を除きます。

  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合であって、本人の同意を得ることが困難なとき
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要がある場合
  • 国の機関もしくは地方公共団体またはその委託を受けた者が法令に定める事務を遂行することに対して協力する必要がある場合

4. 第三者のサービスについて

本アプリでは、広告配信等のために第三者が提供するサービスや SDK を利用しています。 その際、各サービス提供者が利用者情報を取得することがあります。

代表的なサービス例：

  • Google LLC が提供するモバイル広告サービス（Google モバイル広告 SDK / AdMob 等）
  • Expo / React Native 関連のプラットフォームサービス（ビルド・配信基盤など）

これら第三者による情報の取扱いについては、それぞれの事業者が定めるプライバシーポリシーをご確認ください。

5. 個人情報の管理

本アプリの開発者は、利用者からの問い合わせ内容など、メール等で受領した情報について、 不正アクセス・紛失・破壊・改ざん・漏えい等が起こらないよう、適切な安全管理に努めます。

6. 利用者による情報の管理

利用者は、端末の設定画面などから、通知や広告 ID に関する設定を変更・制限することができます。 具体的な操作方法は、ご利用の端末や OS のバージョンによって異なります。

7. 本ポリシーの変更

本プライバシーポリシーの内容は、必要に応じて変更されることがあります。 重要な変更を行う場合は、本アプリ内または公開ページにて告知します。
